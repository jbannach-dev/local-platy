# Loacal Platy

A lightweight open-source tool for local AI text generation using GGUF models.

## Model Setup

To run the application, you need to place an GGUF model file in the internal backend directory.

**Target Path:**
`src-tauri/models/model.gguf`

the path is currently hardcoded

## System Compatibility
Currently, this application has been **developed and tested on Linux with the Qwen3-0.6B Model**.
